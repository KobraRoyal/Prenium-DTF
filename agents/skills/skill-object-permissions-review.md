# Skill — Object Permissions Review

## But
Vérifier qu’aucun utilisateur ne peut accéder à un objet hors de son périmètre.

## Vérifications
- route protégée
- vue protégée
- accès objet filtré
- téléchargement fichier protégé
- tests croisés client A / client B
- tests rôle opérateur / admin / client
