# Override — portails client et opérations

À utiliser avec [`../MASTER.md`](../MASTER.md).

- Style : clair, fonctionnel, haute densité maîtrisée ; statut et action avant décoration.
- Largeur : 1180 px pour les parcours actuels ; grille fluide et cartes empilées sous 768 px.
- Hiérarchie : contexte/breadcrumb → titre/action principale → KPI → file de travail → détail.
- Les tableaux desktop deviennent des cartes lisibles sur mobile quand le scroll masquerait l’action principale.
- Chaque KPI possède un libellé explicite ; chaque statut combine texte et couleur.
- Les panneaux métier partagent onglets, headers, empty states et feedback HTMX.
- Interdits : dashboard sombre par défaut, graphiques sans décision associée, icônes sans libellé et padding arbitraire par vue.
