# Notes d'installation

## ACP recommandé
La doc Goose recommande les ACP providers comme remplaçants des anciens CLI providers dépréciés.
Pour Codex ACP :
1. installer l'adapter ACP
2. authentifier `codex`
3. configurer `GOOSE_PROVIDER=codex-acp`

## CLI providers
Possible en compatibilité, mais la doc indique qu'ils sont dépréciés et n'exposent pas l'écosystème d'extensions Goose.

## Playwright CLI Skill
Pour les recettes UI réelles, installer Playwright CLI si tu veux une recette navigateur plus efficace en coûts qu'un MCP Playwright.
