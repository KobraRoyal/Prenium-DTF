# Prompt — stabilisation avant sprint suivant

Mission :
- analyser les tests rouges
- distinguer test obsolète / contrat cassé / régression
- proposer le correctif minimal
- relancer une suite ciblée
- rendre un verdict prêt / non prêt

Contraintes :
- ne pas ouvrir de nouveau domaine
- respecter DRY / SRP / sécurité / auditabilité
